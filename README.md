This is a plugin that provides communication between WATUT (Fabric) clients on the Bukkit platform.

### Requirements
- Server
    - Minecraft 1.16.5+
    - Bukkit compatible
    - Java 16 <=
- Client
    - Minecraft 1.16.5+
    - Fabric
    - [What Are They Up To (Watut)](https://modrinth.com/mod/what-are-they-up-to)
